Matlab 2015b 运行 EEGlab 出现 ‘excepted input to be…' 问题

2015-12-17
matlab, 学习

在 Matlab 2015b 中运行 EEGLab 后，Matlab 弹出错误对话框，显示内容大概为

解决办法：

这是由于 EEGlab 的BCILab 插件中带的一个叫strjoin.m的函数导致的。由于 Matlab 的执行优先级问题，这个函数代替了 Matlab 自带的同名函数，造成执行错误。将它改名即可。

这个函数位于：

MATLAB\R2015b\toolbox\eeglab13_5_4b\plugins\CleanLine1.03\external\bcilab_partial\dependencies\PropertyGrid-2010-09-16-mod\strjoin2.m

我这里是由于安装了Cleanline这个插件造成的。
